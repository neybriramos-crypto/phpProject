# Proyecto PHP de Gestión de Empleados

Este proyecto es una aplicación web sencilla en PHP para la gestión de empleados.

## Archivos principales

- `dashboard.php`: Página principal después del inicio de sesión.
- `db.php`: Conexión a la base de datos.
- `gestion_empleados.php`: Gestión y administración de empleados.
- `login.php`: Formulario y lógica de inicio de sesión.
- `logout.php`: Cierre de sesión.

## Descripción

La aplicación permite a los usuarios autenticarse y acceder a un panel de control para administrar empleados. Está diseñada para ser un punto de partida básico y puede ampliarse con funcionalidades como registro de usuarios, roles, validación y seguridad adicional.

## Requisitos

- PHP 7.4+ (o superior)
- Servidor web con soporte PHP (XAMPP, WAMP, IIS, Apache, Nginx, etc.)
- Base de datos MySQL/MariaDB

## Uso

1. Configurar el servidor web y colocar el proyecto en el directorio adecuado.
2. Actualizar `db.php` con los datos de conexión a la base de datos.
3. Abrir `login.php` en el navegador para iniciar sesión.
